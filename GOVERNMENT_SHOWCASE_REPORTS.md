### I. Demonstration Architecture Summary
The Government Showcase Mode is a presentation-first environment designed to present the high-impact capabilities of the Iraq Digital Gateway (IDG) platform within a tight 5–10 minute window.

1. **State Engine**: Managed in unified local state with a custom automated progression clock (Chronometer Engine), operating on a tick rate of 1 second and advancing scenes or sub-steps every 5 seconds. This simulates a live, autonomous command center experience suited for exhibition monitors and conference rooms.
2. **Sovereign Layout Grid**: Uses a luxury deep navy/dark layout (`#071739`) inspired by aerospace command structures. It features heavy display headers utilizing responsive negative space, bold visualization cards, custom micro-progress step timelines, and real-time Baghdad UTC clock integration.
3. **Presenter Deck**: Equipped with modular buttons (**Start**, **Pause**, **Resume**, **Next Scenario**, **Prev Scenario**, **Restart**) ensuring the presenter can confidently navigate, pause to elaborate, or jump across scenarios during live briefings.

### III. Executive Storytelling Flow Report
Each phase follows a systematic storytelling framework mapping the flow from operational headache to definitive state victory:

```
[Problem / کێشەی دیاریکراو / المشكلة]
      ↓
[AI Analysis / شیکاری هۆشمەند / التحليل الذكي]
      ↓
[Government Validation / پشتڕاستکردنەوەی حکومی / التدقيق الجمركي الموثق]
      ↓
[Decision Support / یارمەتیدانی بڕیاردان / دعم القرار السيادي]
      ↓
[Outcome / ئەنجامی فەرمی / النتيجة والتحصيل]
```

1. **Problem (المشكلة)**: Introduces the friction point (such as cargo backlogs, overvalued financial declarations, or conflicting tariffs at checkpoint intersections).
2. **AI Analysis (التحليل الذكي)**: System automates classification, identifies pricing anomalies, or parses laws within seconds.
3. **Government Validation (التدقيق الحكومي)**: The platform cross-checks classifications against ministerial decrees and matches citations with official publications to verify national safety and trade rules.
4. **Decision Support (دعم القرار السيادي)**: AI drafts the legal documents, calculates duty levies, and prepares simple choices for directors.
5. **Outcome (المخرج والنتيجة)**: Leverages concrete metrics (reducing delays from 3 days to 32 minutes, protecting $1.2M from flight, or enabling ecological vehicle acquisitions) to prove the system's absolute value to ministers.

### IV. Localization Coverage Report
The Government Showcase Mode is fully purged of any English text, parameters, button names, charts, or indicators. It provides comprehensive dual-localized interfaces supporting top-margin layouts:

- **Kurdish Sorani (Default)**: Optimized using standard vocabulary fit for ministerial cabinets (e.g. "دەستپێکردنی نیشاندانی نیشتمانی", "بنکەی فەرماندەیی نیشتمانی جێبەجێکار").
- **Arabic**: Formulated in formal federal state terminology (e.g. "بدء العرض والتشغيل الوطني", "مركز القيادة الوطني والتحليلات الكبرى").
- **Language Switcher**: Dynamically maps all narratives, step details, alerts, button tags, and status trackers in perfect coordination with the global language selection state inside the `settingsStore`.
